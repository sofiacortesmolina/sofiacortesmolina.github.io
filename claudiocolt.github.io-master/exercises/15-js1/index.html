Desafíos

1. Escribe una función que devuelve un array con todos los números del 1 al 255.

2. Escribe una función que entregue la suma de todos los números pares del 1 al 1000

3. Escribe una función que devuelva la suma de todos los números impares entre 1 y 5000

4. Escribe una función que devuelva la suma de todos los valores dentro de un array

5. Dado un array con múltiples valores, escribe una función que devuelva el número mayor

6. Dado un array con múltiples valores, escribe una función que devuelva el promedio de los valores

7. Escribe una función que devuelva un array de todos los números impares entre 1 y 50 (Usando Push)

8. Dado un valor Y, escribe una función que toma un array y devuelve los valores mayores que Y

9. Dado un array con múltiples valores, escribe una función que reemplace cada valor por el cuadrado del mismo

10. Escribe una función que reemplace cualquier número negativo dentro de un array por 0

11. Devuelve un array que solo contenga el mayor, menor y promedio de un array original

12. Escribe una función que intercambie el primer y el último valor de cualquier array

13. Toma un array de números y reemplaza cualquier valor negativo por el string ‘Dojo’.
